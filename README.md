## Environment Variables

Environment variables are used for ExReddit authentication:

- REDDIT_USER
- REDDIT_PASS
- REDDIT_CLIENT_ID
- REDDIT_SECRET

# Running

After installation, you should just have to do the following, then load localhost:4000 in your browser

    cd apps/summoner_web
    mix phx.server
